# Construction Accounting Automation Portfolio

This repository contains templates and automation tools for accounting and cost management workflows in construction businesses.

## Contents

### 1. Payroll Automation
- Monthly payroll calculation templates
- Variable working hours system support
- Excel and Google Sheets models

### 2. Attendance Data Pipeline
- Attendance data processing scripts (Google Apps Script)
- Monthly aggregation logic

### 3. Cost Management Dashboard
- Project-based cost tracking dashboards
- Visualization templates

### 4. Workflow Documentation
- Accounting workflow diagrams
- Error prevention design concepts
